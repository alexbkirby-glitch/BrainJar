Privacy Policy

Last updated: June 2026 · Applies to alexbkirby-glitch.github.io/lodestone

Lodestone is a client-side-only tool. It has no server, no database, and sets no cookies. All processing happens in your browser.

Who we are

Lodestone is an open-source project maintained by Alex Kirby and the community. Source code: github.com/alexbkirby-glitch/lodestone. For privacy-related questions, please open an issue on GitHub.

What data we collect

Lodestone itself collects no personal data. We do not use analytics, do not set cookies, and do not log any user activity.

The following data is stored locally in your browser (localStorage) when you use certain features:

  • API keys — Anthropic, OpenAI, and GitHub tokens you enter are stored in your browser's localStorage. They do not leave your device except when used to make API calls to the respective provider.
  • User preferences — UI settings and session state.

You can clear all locally stored data at any time using your browser's Developer Tools (Application → Local Storage → Clear) or via the Settings panel in the app.

Third-party services

When you use Lodestone's AI-powered features, your intent text and API key are sent directly from your browser to the following providers:

  • Anthropic — when using Claude models. See Anthropic's privacy policy. API data is not used for training by default.
  • OpenAI — when using OpenAI models. See OpenAI's privacy policy.
  • GitHub — when using the Publisher or StoneHub features. Your GitHub Personal Access Token is used only to create files in repositories you specify. See GitHub's privacy statement.
  • Google Fonts — fonts are loaded from Google's servers. See Google's privacy policy.

Lodestone does not sell, share, or broker any user data. We have no visibility into what data you send to third-party APIs.

Hosting

This site is hosted on GitHub Pages. GitHub may collect server logs including IP addresses as part of operating the hosting service. See GitHub's privacy statement.

Your rights

You can delete all data Lodestone stores by clearing your browser's localStorage. For data held by third-party providers (Anthropic, OpenAI, GitHub, Google), refer to their respective privacy policies and data deletion procedures.

Changes to this policy

This policy may be updated. The "last updated" date at the top reflects the most recent revision. Significant changes will be noted in the project's changelog.
